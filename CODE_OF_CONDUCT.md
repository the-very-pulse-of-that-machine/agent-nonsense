# Code of Conduct

Agent Nonsense contributors are expected to keep project spaces respectful, constructive, and focused on the work.

- Be considerate when reviewing code and discussing tradeoffs.
- Critique ideas and implementations, not people.
- Do not harass, threaten, discriminate against, or expose private information about others.
- Respect maintainers' decisions about scope, safety, and release quality.

Project maintainers may edit or remove contributions and restrict participation when behavior undermines a safe and productive community. Report conduct concerns privately to the maintainers rather than escalating them in public issues.
